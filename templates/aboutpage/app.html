About

eX-Opti is a multi-platform expense management system built on the cloud and hosted on Google App Engine. It can be accessed either through a PC or a mobile web browser or through telephony.

It currently offers the following services:
1) Adding new transaction entries.
2) Retrieving analytics of previous spendings
3) Making Transactions (waiting for ZAP API to complete implementation)

It uses Kookoo API for telephony support and Google Charts API for displaying analytics.
